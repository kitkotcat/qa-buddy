# Политика конфиденциальности QA Cat Buddy

**Дата обновления: 19 июня 2026 года**

## 1. Общая информация

QA Cat Buddy — приложение для практики и самопроверки знаний в области тестирования программного обеспечения. Для использования приложения не требуется регистрация или создание учётной записи.

## 2. Локальные данные

Приложение может сохранять на устройстве:

- выбранный язык;
- цель подготовки;
- прогресс и результаты тестов;
- состояние чек-листов;
- созданные баг-репорты и тест-кейсы;
- настройки приложения.

## 3. Передача данных

Android-версия QA Cat Buddy не передаёт сохранённый прогресс, настройки и созданные материалы разработчику или третьим лицам. Основные функции доступны без подключения к интернету.

## 4. Персональные данные

Приложение не запрашивает имя, номер телефона, адрес электронной почты, дату рождения, местоположение, контакты, фотографии, аудиозаписи или платёжные данные пользователя.

## 5. Разрешения

Приложение не предназначено для получения доступа к камере, микрофону, контактам, сообщениям, журналу вызовов или геолокации.

Фактический перечень разрешений проверяется перед публикацией каждой версии.

## 6. Удаление данных

Локальные данные можно удалить через настройки Android, очистив данные приложения, либо удалив приложение с устройства.

## 7. Использование детьми

Приложение не содержит общения между пользователями, рекламы, покупок, азартных игр или контента для взрослых и не осуществляет целевой сбор данных детей.

## 8. Изменения политики

Политика может обновляться при изменении функций приложения, способов обработки данных или требований законодательства.

## 9. Контакты

Связаться с разработчиком можно по контактному адресу, указанному в карточке QA Cat Buddy в RuStore.
