# QA Buddy v0.3.0 — Android Offline MVP

QA Buddy is now available as an Android offline application.

## Added

* Android application powered by Capacitor
* Offline checklists and interview questions
* Offline Bug Report and Test Case generators
* QA Knowledge Quiz with 5, 10 and 20 question modes
* Quiz progress, category statistics and mistake review
* Learning-goal onboarding
* Personalized QA Cat illustrations
* Russian and English interface
* Persistent language selection
* Mobile-responsive navigation
* Improved Interview Trainer:

  * hidden answers by default
  * Show Answer action
  * short and detailed answers
  * smooth mobile scrolling
  * next-question navigation
